# E-Learning Website using HTML5, CSS3, Bootstrap5 and JavaScript 

![E-Learning Website](preview.jpg)
CoderPathway is a modern and responsive e-learning website developed using HTML5, CSS3, Bootstrap 5, and JavaScript.
The platform is designed to provide a simple, interactive, and visually appealing environment where users can explore courses and begin their learning journey.


## Introduction
CoderPathway is an educational web platform that aims to make learning more accessible and engaging. It provides users with structured course information, smooth navigation, and a responsive design that works across all devices.

The project focuses on frontend development and demonstrates how modern web technologies can be used to build a professional e-learning interface.
## Features

- Responsive design using Bootstrap 5.
- Structured course catalog with detailed descriptions.
- Interactive lessons.
- User can register and login.
- User authentication for personalized learning experiences.
- Instructor application form.
- User can also contact us.
